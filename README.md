# Bedrock

## Requirements

- node.js 8+

## Starting

```bash
npx create-bedrock my-design-system
cd my-design-system
npm install
npm start
```

Open these browser windows:

- http://localhost:3999

## Commands

- `npm start` - Compile, start all watches and local server
- `npm run build` - Compile with compression turned on
- `npm run serve` - Start server
